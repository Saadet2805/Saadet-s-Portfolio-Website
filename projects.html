My Projects

Web Design and Development using HTML, PHP and SQL (front and back end): Profile Management System

GitHub Link

This is a project I made for my CEN311 Web Technologies and Programming Course in my second year of studies. In this course we took HTML, PHP and MySQL and learned both front end and back end. This project was a lab project to store information about the users of an application. In this website you can login, add, edit and delete users. All the information is displayed on the main page and you can get directed to other pages by the sidebar. I used the hashing algorithm to store the passwords and a bootstrap template for the overall design.

Programming using C: Car Rental Management System

GitHub Link

This is a group project we did in our first year of studies for the CEN 110 C Programming Course. In this project, we made an interactive system through C for a car rental management system. First a menu is displayed in the command prompt and the manager/staff of the company can choose the action they want to take. They can: display cars sorted according to their price etc., search for cars by their plate number, display reservations, edit and delete them and many other operations.
